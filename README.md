# Movie Recommendation System

A small web app that suggests movies based on one you already like. You pick a movie from the list, and the app returns a set of similar movies. It looks at the movie's own details (genres, keywords, cast, and plot) and at how other users have rated movies, and combines the two to make the suggestions.

The project is still in progress.

## Planned stack

- Backend and API: Python with FastAPI
- Frontend: React
- Database: PostgreSQL for the movie catalog
- Everything runs in Docker with docker compose

## Dataset

The Movies Dataset from Kaggle (by Rounak Banik). It brings together movie details from TMDb and user ratings from MovieLens.

Link: https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset
